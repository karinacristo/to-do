# Todo API

Esta é uma API de gerenciamento de tarefas (To-Do List) construída com Node.js, Express, MongoDB, JWT e bcryptjs.

## Tecnologias Utilizadas

- Node.js
- Express.js
- MongoDB
- JWT (JSON Web Tokens)
- bcryptjs
- dotenv
